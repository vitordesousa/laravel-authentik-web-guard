

#  Authentik Web Guard for Laravel (Inspired by Laravel Keycloak Web Guard)

This packages allow you authenticate users with [Authentik Server](https://www.authentik.org).

## Requirements

* Have a Authentik Server.
* Have a realm configured and a client that accepts authentication.

### Support

This package was tested with:

* Laravel: 5.8 / 7 / 8 / 9 / 10 / 11 / 12
* Authentik: 18.0.0+
